সমস্যা সমাধান
Problem Solving
০১

বাস্তবজীবনে আমরা নানান ধরনের সমস্যার সম্মুখীন হই। সমস্যা সমাধান হল চ্যালেঞ্জ বা বাধা শনাক্ত করার প্রক্রিয়া, যাতে সমস্যার প্রকৃতি বোঝা যায় এবং এটি অতিক্রম করার উপায় খুঁজে বের করা যায়।

বাস্তব উদাহরণ: পরিবেশ পরিস্থিতি বিশ্লেষণ করা, চিন্তা করা এবং সমস্যার সম্ভাব্য সমাধান খুঁজে বের করে নিয়ে আসার বিষয়টি জড়িত। এই প্রক্রিয়ায় সাধারণত সমস্যা সংজ্ঞায়িত করা, সমস্যার সম্ভাব্য সমাধান নিয়ে চিন্তাভাবনা করা, সেই বিকল্প সমাধানগুলো মূল্যায়ন করা এবং তারপরে সেরা সমাধানটি বাস্তবায়ন করা অন্তর্ভুক্ত থাকে।
সমস্যা সমাধানের ধাপসমূহ (পাঠ্যপুস্তক অনুযায়ী)
১. সমস্যা সংজ্ঞায়িত করা ২. পরিবেশ পরিস্থিতি বিশ্লেষণ করা ৩. সম্ভাব্য সমাধান নিয়ে চিন্তাভাবনা করা ৪. বিকল্প সমাধানগুলো মূল্যায়ন করা ৫. সেরা সমাধান বাস্তবায়ন করা
কম্পিউটারে সমাধান করতে হলে প্রথমে প্রোগ্রামিং ভাষায় নির্দেশ লিখতে হয়। আমরা যেকোনো একটি প্রোগ্রামিং ভাষা শিখলে সেই ভাষার মাধ্যমে কম্পিউটারকে প্রয়োজনমতো বিভিন্ন নির্দেশ দিতে পারব।
মনে রাখবে

আমাদের কম্পিউটারের একটি সফটওয়্যার বা অ্যাপ্লিকেশনে প্রথমে আমরা নির্দিষ্ট কোনো প্রোগ্রামিং ভাষায় নির্দেশগুলো লিখে ফেলব। কম্পিউটারে এমন একটি রূপান্তর ব্যবস্থা থাকে যা সেই প্রোগ্রামিং ভাষার নির্দেশগুলোকে মেশিন কোডে রূপান্তর করে।

নিজে করো
  1. "দুটি সংখ্যার মধ্যে বড়টি খুঁজে বের করো" — সমস্যা সমাধানের ধাপগুলো লেখো।
  2. তোমার দৈনন্দিন জীবনের যেকোনো একটি সমস্যার সমাধান ধাপে ধাপে লেখো।
প্রোগ্রামিং ভাষা
Programming Language
০২

আমরা মনের ভাব প্রকাশের জন্য বিভিন্ন ভাষা ব্যবহার করি যেমন— বাংলা, ইংরেজি, ফ্রেঞ্চ, ল্যাটিন, স্প্যানিশ ইত্যাদি। কিন্তু কম্পিউটার আমাদের এ সকল ভাষা সরাসরি বুঝতে পারে না। কম্পিউটারকে যেকোনো নির্দেশ দিতে গেলে কম্পিউটার বুঝতে পারে এমন ভাষায় নির্দেশ লিখতে হয় — এই ভাষাগুলোকেই বলা হয় প্রোগ্রামিং ভাষা।

প্রচলিত প্রোগ্রামিং ভাষা: C, C++, Python, Java, JavaScript, PHP, Ruby, Objective-C, Perl, Visual Basic — এরকম অনেক প্রোগ্রামিং ভাষা আছে।
গুরুত্বপূর্ণ তথ্য
• কম্পিউটার শুধু ০ এবং ১ বুঝতে পারে • মানুষের ভাষাকে ০/১ এ রূপান্তর করতে প্রোগ্রামিং ভাষা ব্যবহার হয় • এই রূপান্তরের কাজ করে "অনুবাদক" প্রোগ্রাম • সব প্রোগ্রামিং ভাষার মূল গঠন একই রকমের
বর্তমানে প্রচলিত প্রোগ্রামিং ভাষাগুলোর মধ্য থেকে যেকোনো একটি প্রোগ্রামিং ভাষা প্রথমে শিখলেই হবে। কারণ সব প্রোগ্রামিং ভাষার মূল গঠন একই রকমের। তুমি একটি প্রোগ্রামিং ভাষা শিখে নিলে এরপর তোমার জন্য অন্য প্রোগ্রামিং ভাষাগুলো শেখা খুবই সহজ হয়ে যাবে।
মনে রাখবে

প্রোগ্রামিং যারা নতুন শিখতে চায় তাদের জন্য Python একটি দুর্দান্ত সূচনা হতে পারে। কারণ Python শিক্ষাবান্ধব, সহজে শেখার জন্য এটির একটি সাধারণ সিনট্যাক্স রয়েছে।

নিজে করো
  1. পাঁচটি প্রোগ্রামিং ভাষার নাম লেখো।
  2. Python কেন নতুনদের জন্য উপযুক্ত — দুটি কারণ লেখো।
মেশিন কোড
Machine Code
০৩

মূলত ০ আর ১ এর সমন্বয়ে তৈরি বাইনারি কোডকেই মেশিন কোড বা মেশিন ভাষা বলা হয়, যা আমাদের কম্পিউটার সরাসরি বুঝতে পারে। কাজেই প্রোগ্রামিং ভাষা ও মেশিন কোডের মধ্যে রূপান্তরের কাজটি অনুবাদক নামক এক ধরনের প্রোগ্রামের সাহায্যে করা হয়ে থাকে।

উদাহরণ: HELLO শব্দটি মেশিন কোডে → 01001000 01000101 01001100 01001100 01001111 এভাবে ০ এবং ১ দিয়ে প্রকাশ পায়।
মেশিন কোডের বৈশিষ্ট্য
• শুধুমাত্র ০ এবং ১ দিয়ে তৈরি (Binary) • কম্পিউটার সরাসরি বুঝতে পারে • মানুষের পক্ষে সরাসরি লেখা কঠিন • প্রোগ্রামিং ভাষা → অনুবাদক → মেশিন কোড
রূপান্তরের ধারা
প্রোগ্রামিং ভাষা → TRANSLATOR → মেশিন কোড
অনুবাদকের মাধ্যমে মেশিন কোডে রূপান্তরের ফলে আমাদের নির্দেশগুলো কম্পিউটার বুঝতে পারবে এবং সেই নির্দেশ অনুসরণ করে নির্ধারিত কাজ সম্পন্ন করতে পারবে।
মনে রাখবে

কম্পিউটারসহ যেকোনো ইলেকট্রনিক ডিভাইস শুধু ০ আর ১ কে বুঝতে পারে। কিন্তু শুধু ০ আর ১ দিয়ে নিজেদের নির্দেশগুলো লিখে ফেলা আমাদের জন্য কঠিন।

নিজে করো
  1. মেশিন কোড কাকে বলে? নিজের ভাষায় লেখো।
  2. প্রোগ্রামিং ভাষা থেকে মেশিন কোডে রূপান্তর কীভাবে হয় — ব্যাখ্যা করো।
কম্পাইলার
Compiler
০৪

কিছু রূপান্তর ব্যবস্থায় আমরা যতগুলো নির্দেশই দিই, যদি নির্দেশগুলো নির্ভুল হয় তাহলে সবগুলো নির্দেশ একসাথে মেশিন কোডে রূপান্তরিত হবে। এই রূপান্তর ব্যবস্থাকে বলা হয় কম্পাইল (Compile) করা। আর যে সফটওয়্যার রূপান্তর করল, সেই রূপান্তরকারী প্রোগ্রামটি হচ্ছে একটি কম্পাইলার (Compiler)।

Python-এর গাণিতিক অপারেটর (পাঠ্যপুস্তক)
+ → যোগ (Addition) a + b - → বিয়োগ (Subtraction) a - b * → গুণ (Multiplication) a * b / → ভাগ (Division) a / b (ফলাফল float) % → ভাগশেষ (Modulo) a % b উদাহরণ: a=5, b=2 print(a+b) # 7 print(a-b) # 3 print(a*b) # 10 print(a/b) # 2.5 print(a%b) # 1

Comparison অপারেটর (পাঠ্যপুস্তক সারণি)
== Equal Op1==10 → সমান হলে True != Not Equal Op1!=10 → অসমান হলে True < Less Than Op1<10 → ছোট হলে True <= Less or Equal Op1<=10 → ছোট বা সমান হলে True > Greater Than Op1>10 → বড় হলে True >= Greater/Equal Op1>=10 → বড় বা সমান হলে True
মনে রাখো: = একটি (assignment) মানে মান রাখা। == দুটি (comparison) মানে তুলনা করা।

if statement এর গঠন
if condition: statement(s) to be executed if condition is true # উদাহরণ (পাঠ্যপুস্তক - ব্যবহারিক সমস্যা ২): mark = int(input('Please enter your mark: ')) if mark >= 40: print('You have passed') if mark < 40: print('You have failed')
if statement কাজ করে যেভাবে:
• শর্তের মান সত্য হলে → if এর বডি চলবে

• বডির কোড ইনডেন্ট (4 স্পেস) করতে হবে

if...else এর গঠন ও উদাহরণ
if Condition: statements ← Body of if else : statements ← Body of else # উদাহরণ ৩ (ভোটার যাচাই): age = int(input('Please enter your age: ')) if age >= 18: print('You are eligible for voting') else : print('You are not eligible for voting') # উদাহরণ ৪ (জোড়/বিজোড়): num = int(input('Insert a number= ')) if num%2!= 0: print('The number is odd') else : print(' The number is even')
if...else কীভাবে কাজ করে:
• Condition সত্য → Body of if চলবে
• Condition মিথ্যা → Body of else চলবে
• দুটির মধ্যে একটিই সবসময় চলবে
উল্লেখ্য যে, if...else কন্ট্রোলে ব্যবহৃত শর্ত (Condition) সাধারণত এক বা একাধিক লজিক্যাল বা রিলেশনাল এক্সপ্রেশন হয়। if (Condition) স্টেটমেন্টের পরে কোলন বসে।
মনে রাখবে

এই Condition বা শর্তের মান যদি সত্য হয় তবে Body of if এ বর্ণিত কাজগুলো সম্পাদিত হবে। আর যদি মিথ্যা হয় তাহলে Body of else এ বর্ণিত কাজসমূহ সম্পাদিত হয়।

১৬

একাধিক শর্ত যাচাই করার জন্য পাইথন প্রোগ্রামে "অন্যথায় যদি" অর্থে if...else স্টেটমেন্টের সাথে elif স্টেটমেন্ট ব্যবহার হয়। elif স্টেটমেন্ট else এবং if এর মাঝে থাকে।

elif এর গঠন ও উদাহরণ
if (Condition 1): Block1 elif (Condition 2): Block2 ... else : DefaultBlock # উদাহরণ ৭ (ধনাত্মক/ঋণাত্মক/শূন্য): num = int(input('Insert a number: ')) if num > 0: print('The number is positive') elif num < 0: print('The number is negative') else : print('It is zero')

১৭

পাইথন প্রোগ্রামে elif স্টেটমেন্টের অনুরূপ কাজ অর্থাৎ একাধিক স্টেটমেন্ট থেকে নির্দিষ্ট কোনো স্টেটমেন্ট নির্বাহের জন্য match স্টেটমেন্ট ব্যবহার হয়। মূলত বেশি সংখ্যক elif স্টেটমেন্ট ব্যবহারের পরিবর্তে match স্টেটমেন্ট ব্যবহার হয়।

match এর গঠন ও উদাহরণ
match MatchExp: case Value1 : Block1 case Value2 : Block2 case _ : DefaultBlock # উদাহরণ ৮ (Vowel/Consonant - পাঠ্যপুস্তক): char = input('Enter the letter: ') match char: case "a" : print(char, ' is a vowel ') case "e" : print(char, ' is a vowel ') case "i" : print(char, ' is a vowel ') case "o" : print(char, ' is a vowel ') case "u" : print(char, ' is a vowel ') case _ : print(char, ' is a consonant')
match স্টেটমেন্টের বিশেষত্ব:
• elif কন্ডিশনাল এক্সপ্রেশনের উপর ভিত্তি করে কাজ করে
• match সাধারণত variable-এর মানের ভিত্তিতে কাজ করে
• case _ হলো ডিফল্ট (যখন কোনো case মিলবে না)
match স্টেটমেন্টের সাথে ব্যবহৃত এক্সপ্রেশনকে ম্যাচ এক্সপ্রেশন (MatchExp) বলা হয়। case কীওয়ার্ড ও ম্যাচ এক্সপ্রেশন ভেরিয়েবলের সম্ভাব্য মানের মাঝে ন্যূনতম একটি স্পেস থাকে এবং কোলন দ্বারা শেষ হয়।

for loop এর গঠন ও উদাহরণ
for CounterInitialization in Condition : // Statement(s) # উদাহরণ ১০ (ICT ৫ বার): for i in range(1,6): print('ICT') # বিকল্প (Increment by 1): for i in range(1,6,1): print('ICT') # বিকল্প (Decrement by 1): for i in range(6,1,-1): print('ICT')
ব্যবহারিক সমস্যা ১২ (পাঠ্যপুস্তক)
# ১ থেকে ১০০ পর্যন্ত বিজোড় সংখ্যা বের করা: for i in range(1, 101): if (i % 2 != 0): print(i) # গুণের টেবিল (সমস্যা ১৪): num = int(input('Enter a number: ')) for i in range(1,11): prod = num*i print(num, '*', i, '=', prod)
range() ফাংশন বোঝা:

range(1,6,1) → 1,2,3,4,5 (1 করে বাড়ে)
range(6,1,-1) → 6,5,4,3,2 (1 করে কমে)

# continue স্টেটমেন্ট প্রোগ্রাম পয়েন্টারকে পূর্ববর্তী # স্টেটমেন্ট বা লুপের প্রান্তে স্থানান্তর করে। # উদাহরণ ১৭ (পাঠ্যপুস্তক - ৪ বাদ দিয়ে ১-১০): for num in range(1, 11, 1): if num == 4: continue else: print(num, end=" ") print('Used continue to skip printing the value 4') # আউটপুট: 1 2 3 5 7 8 9 10 # Used continue to skip printing the value 4
continue কীভাবে কাজ করে:
• শর্ত সত্য হলে continue কার্যকর হয়
• বর্তমান iteration এর বাকি অংশ বাদ দিয়ে
• পরের iteration এ চলে যায়
• লুপ বন্ধ হয় না, শুধু ওই ধাপটি এড়িয়ে যায়

if / else / elif
CONTROL · ৬.৩

                
✓

              

              
if
শর্ত সত্য হলে → if এর body চলবে।

              
else
শর্ত মিথ্যা হলে → else এর body চলবে।

              
elif
একাধিক শর্ত যাচাইয়ের জন্য ব্যবহার হয়।

              
age = int(input("বয়স: "))
if age >= 18:
    print("ভোটার")
elif age >= 13:
    print("কিশোর")
else:
    print("শিশু")

            

          


          
          

            

            

              

                

                
match statement
CONTROL · Switch-Case

                
✓

              

              
→
match ভেরিয়েবলের মানের উপর ভিত্তি করে block চালায়।

              
_
case _ হলো ডিফল্ট — কোনো case না মিললে চলে।

    case _:
        print("ব্যঞ্জনবর্ণ")

            

          


          
          

            

            

              

                

                
for লুপ
LOOP · ৬.৪

→
